# FocusTerminal

FocusTerminal es una aplicación de consola desarrollada en C# para registrar, organizar y consultar tareas académicas mediante programación estructurada y archivos de texto.

## Objetivo

Apoyar la organización académica de estudiantes universitarios mediante un gestor de tareas local, ligero y fácil de usar desde la terminal.

## Características principales

- Registro de tareas académicas.
- Edición y eliminación de tareas.
- Cambio de estado de tareas.
- Filtro por estado, prioridad o categoría.
- Visualización de estadísticas generales.
- Guardado y carga de datos mediante archivo de texto.

## Tecnologías utilizadas

- C#
- .NET
- Aplicación de consola
- Archivos de texto
- Git y GitHub
